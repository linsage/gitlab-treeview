# GitLab-TreeView

> fork from [gitlab-treeview](https://github.com/linsage/gitlab-treeview) and fix some bug

Chrome extension to show a code tree on GitLab :)

## Features

- Quick preview code
- GitLab UI style
- Search File
- Support gitlab 9.x version
